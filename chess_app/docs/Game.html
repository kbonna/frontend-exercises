canCastle :Object

Stores information about possible castling for each side. Castling is permitted if both king and selected rock did not move during the game. Additional conditions are checked during actual move by getCastleMoves method. Note that first array element corresponds to long castle (rock on column 0), and second one to short castle (rock on column 7).
